# Web Store

**Web Store** is a marketplace for PWA applications where users can discover and install progressive web apps. Web Store itself is also a PWA and can be installed on devices.

## Description

Web Store is an App Store/Play Market equivalent for web applications. The platform enables:
- **Users**: to discover, test, and install PWA applications like native apps
- **Developers**: to easily publish their PWAs, get analytics, and monetize applications

## Repositories

The project consists of three main repositories:

1. **Frontend** — main repository with React application
2. **Backend** — API and server logic (embedded in frontend via Supabase)
3. **Docs** — public documentation (this repository)

## Technologies

- **Frontend**: React 18, TypeScript, Vite
- **State Management**: Jotai, Apollo Client (GraphQL)
- **Styling**: Tailwind CSS, shadcn/ui
- **PWA**: Vite PWA Plugin, Service Workers
- **Backend**: PostgreSQL, GraphQL, Auth, Storage
- **Payments**: PayPal, YooMoney
