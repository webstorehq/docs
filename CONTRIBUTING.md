# Contributing to Web Store

Thank you for your interest in Web Store! This guide explains how to report issues and communicate with the project.

## Reporting Issues

### Bug Report

When reporting a bug, please include:

```markdown
## Bug Description
Brief description of the problem

## Steps to Reproduce
1. Go to '...'
2. Click on '...'
3. Scroll down to '...'
4. See error

## Expected Behavior
What should have happened

## Actual Behavior
What actually happened

## Screenshots
If applicable

## Environment
- Browser: [e.g. Chrome 120]
- OS: [e.g. macOS 14]
- Version: [e.g. 1.0.0]

## Additional Information
Any other relevant information
```

### Feature Request

When requesting a feature, please include:

```markdown
## Feature Description
What you want to add

## Problem This Solves
What problem does this feature solve

## Proposed Solution
How you envision the implementation

## Alternatives
Other possible solutions

## Additional Information
Any other relevant information
```
